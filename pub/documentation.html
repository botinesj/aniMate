aniMate.js
A learning utility library that allows users to easily create animations representing plays or events in sports.
Home Documentation
Getting Started
The following code must be included in the head of your HTML file:
<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.6.0/jquery.min.js"></script>
<script defer src="https://cdnjs.cloudflare.com/ajax/libs/jqueryui/1.12.1/jquery-ui.min.js"></script>
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/jqueryui/1.12.1/jquery-ui.min.css">
<script defer src="https://cdn.jsdelivr.net/npm/lodash@4.17.21/lodash.min.js"></script>
<script defer type="text/javascript" src='js/animate.js'></script>
How to Use
Create the HTML Div element that will be the animation field:
<div id="animationField"></div>
The HTML Div element must be relatively positioned, otherwise the library will not work:
#animationField {
...
position: relative;
...
}
Then use the library in the JavaScript file. The background colours for the player and ball elements must be among the following: Black, Red, Orange, Yellow, Green, Blue, Purple or Brown. Please note that the order of the aniMate method calls must be the same as below for the library to work:
const aMlibrary = new aM() // Create the object necessary to use the library

// Create objects that you want to be animated
const player = document.createElement('div')
player.style = 'width: 40px; height: 40px; border-radius: 50%; background-color: Blue;'

const ball = document.createElement('div')
ball.style = 'width: 30px; height: 30px; border-radius: 50%; background-color: Brown;'

// Create the horizontal interacter element
const hInteracter = aMlibrary.horizontalInteracter()

aMlibrary.initiate('#animationField') // Set the HTML Div element that we want to be the animation field
aMlibrary.setReset(true) // Set whether we want the animations to reset upon completion
$('#animationField').after( hInteracter); // Insert the horizontal interacter into the DOM wherever desired
aMlibrary.setSlider(); // Set the slider in the horizontal interacter
aMlibrary.setPlayer(player) // Set the HTML element that will represent a player
aMlibrary.setBall(ball) // Set the HTML element that will represent a ball
Methods
const aMlibrary = new aM()
Description: new aM() returns the object necessary to use the library.
aMlibrary.initiate(input)
Description: Sets the input, a specific HTML element, to be the current animation field (where player as well as ball elements can be placed and animated within by the user). The function also deletes any player or ball elements from a previously used animation field that is located on the same page.
Input: The input to this function is the selector query of the desired animation field, which must be a Div object.
aMlibrary.setReset(input)
Description: This function sets the private reset variable to be true or false. If the input is true, then whenever all the animations in the current animation field finish running, the elements within the animation field will return to their starting positions prior to the "Play Animations‚Äù button being clicked clicked initially, with the exact same animation paths. If the input is false, then whenever all the animations in the current animation field finish running, the elements will remain in the final position of their animation path and their previous animation paths will be erased.
Input: The input to this function is a boolean (true or false).
aMlibrary.horizontalInteracter()
Description: This function returns an HTML Div element, known as the horizontalInteracter, which the user can interact with to perform additional functionality on the current animation field. The developer can then place this HTML Div element wherever they desire relative to the current animation field.
aMlibrary.setSlider()
Description: This function creates the slider within the horizontalInteracter, and initializes its (the slider's) functionality.
aMlibrary.setPlayer(input)
Description: This function sets the player element, which is to be used in an animation field, to be the given HTML element.
Input: The input to this function should be the actual HTML element (NOT its selector query).
aMlibrary.setBall(input)
Description: This function sets the ball element, which is to be used in an animation field, to be the given HTML element.
Input: The input to this function should be the actual HTML element (NOT its selector query).
